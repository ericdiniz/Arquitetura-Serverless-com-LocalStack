# Lab 04 - Serverless com LocalStack

Projeto de exemplo para o roteiro "Arquitetura Serverless com LocalStack".

Estrutura inicial criada automaticamente.

Comandos úteis:

```bash
# Instalar dependências (local)
cd lab04-serverless-localstack
npm install

# Iniciar LocalStack (docker-compose)
npm run docker:up

# Deploy (Serverless + LocalStack)
npm run deploy
```
